# МатЛэнд — 110 игр по математике для 1–11 классов

Все игры считаются в браузере (~50 КБ кода, ни одной картинки) — сайт не тормозит ни при какой нагрузке и работает даже на слабых телефонах и медленном интернете. Лёгкий Node-сервер нужен только для рейтинга игроков.

## Что внутри

- **11 классов × 10 игр** = 110 игр (7 бесплатных + 3 премиум в каждом классе)
- **15 механик**: гонка, ловец, шары, память, башня, дуэль, спринт, космос, числовая прямая, порядок, верно/неверно, рыбалка, босс-битва, тропа, пазл
- Задачи **генерируются бесконечно** по программе каждого класса — игры не надоедают
- **Рейтинг по каждой игре**: топ-20 игроков, свою позицию видно даже за пределами топа. Игрок заполняет анкету (имя, фамилия, город, школа, класс) — кнопка «Профиль»
- **Призы**: первые три места в рейтинге игры автоматически видят промокод на премиум (по умолчанию `CHAMPION`, меняется переменной окружения `REWARD_CODE`)
- Очки: 100 за каждый верный ответ + 250 за каждую звезду + бонус за скорость (до 600, только при результате от 1 звезды); время партии видно на экране игры, в итогах и в таблице рейтинга; в рейтинг идёт лучший результат
- Темп подстроен под возраст: в 1 классе ~7 секунд на задачу, в 11 — ~18; в аркадах ответы не исчезают навсегда, а возвращаются по кругу, пока не решишь
- Прогресс и звёзды сохраняются на устройстве (localStorage)
- Премиум открывается промокодом: `MATLAND2026`, `SCHOOL110`, `TEACHER` или призовым `CHAMPION`. Кнопка «Купить» — заглушка: подключите ЮKassa/Stripe, когда будете готовы принимать оплату.

## Запуск локально

```bash
cd mathgames
npm install
npm start          # http://localhost:3000
```

Можно открыть и просто `index.html` без сервера — всё будет работать, кроме рейтинга.

## Деплой на Railway (рекомендуется — сайту теперь нужен сервер для рейтинга)

1. Запушьте код на GitHub:
   ```bash
   cd mathgames
   git init && git add -A && git commit -m "МатЛэнд"
   git remote add origin https://github.com/ВАШ_ЛОГИН/matland.git
   git branch -M main && git push -u origin main
   ```
2. На [railway.app](https://railway.app): **New Project → Deploy from GitHub repo** → выберите репозиторий. Railway сам выполнит `npm install` и `npm start`.
3. **Settings → Networking → Generate Domain** — получите публичный адрес.
4. Чтобы рейтинг переживал редеплой: добавьте **Volume** (например, в `/data`) и переменную окружения `DB_PATH=/data/db.json`.
5. (Опционально) переменная `REWARD_CODE=ВАШКОД` — свой призовой промокод для топ-3.

## Оплата премиума

Кнопка «Купить» предлагает два способа. Оба включаются переменными окружения на Railway (Settings → Variables):

**СБП / карта через ЮKassa:**
- `YOOKASSA_SHOP_ID` — shopId из личного кабинета ЮKassa
- `YOOKASSA_SECRET` — Secret Key оттуда же
- `PREMIUM_PRICE` — цена в рублях (по умолчанию `199.00`)
- `SITE_URL` — публичный адрес сайта (для возврата после оплаты)
- В кабинете ЮKassa включите вебхук: URL `<ваш сайт>/api/pay/webhook`, событие `payment.succeeded`. После оплаты премиум привязывается к устройству покупателя автоматически.

**Telegram-бот (оплата в Telegram Stars, платёжный провайдер не нужен):**
- Создайте бота у @BotFather, получите токен
- `BOT_TOKEN` — токен бота (бот стартует автоматически вместе с сервером)
- `TG_BOT_URL` — ссылка на бота, например `https://t.me/matland_bot` (появится кнопкой на сайте)
- `UNLOCK_CODE` — промокод, который бот выдаёт после оплаты (по умолчанию `CHAMPION`)
- `STARS_PRICE` — цена в звёздах (по умолчанию `100`)

Пока переменные не заданы, кнопки честно сообщают «оплата подключается» и предлагают промокод.

Альтернатива — [render.com](https://render.com) (Web Service, есть бесплатный тариф, но сервис засыпает без трафика). GitHub Pages для этой версии не подходит: там нельзя запустить сервер рейтинга.

## Структура

```
index.html      — разметка всех экранов
css/style.css   — дизайн (тёмная тема, адаптив под телефон и ПК)
js/problems.js  — генераторы задач по программе 1–11 классов
js/engine.js    — 15 игровых механик (canvas + DOM, без библиотек)
js/data.js      — каталог 110 игр
js/app.js       — роутинг, прогресс, премиум, анкета и рейтинг
server.js       — сервер: статика + API рейтинга (Express, хранение в JSON)
```

Промокоды меняются в `js/app.js` (константа `PROMO_CODES`).
